Useful and Recommended Software

These applications are listed here for your convenience only. They are not endorsed nor supported by Feral Hosting. We will try to answer simple questions about use, but ultimately the support for each application falls to the software developer. If an application does not do what you expect it to, please look on their website, forum or knowledge-base.

Some applications gave a portable version or PortableApps version. Portable apps can be run from a USB stick and do not write any configs to the local PC. Carry your apps in your pocket! There are lots more portable apps here.

Windows


Bitkinex - BitKinex is a handy network client supporting multiple transfer protocols (FTP, FTP(S), SFTP, HTTP(S) and WebDAV). Supports segmented downloads - Full Features list

WinSCP - Free SFTP and FTP client for Windows. Supports custom commands.
Also available - WinSCP Portable

FileZilla - Free FTP client that runs on  all major platforms. Supports transferring multiple files at the same time.
Also available - FileZilla Portable.

CuteFTP Home or Pro - paid multi-file and multi-part FTP client for Windows and Mac.

Aria2c - aria2 is a lightweight multi-protocol & multi-source command-line download utility. It supports HTTP/HTTPS, FTP, BitTorrent and Metalink. aria2 can be manipulated via built-in JSON-RPC and XML-RPC interfaces.

XShell SSH client - Free for home use - Xshell is a powerful terminal emulator that supports SSH, SFTP, TELNET, RLOGIN and SERIAL.

PuTTy - Free SSH client for Windows. For Macs, you can just use the built-in terminal or xterm. Or see below for link to Mac version of Putty.
Also available - Putty Portable.

kitty - KiTTY is a fork from version 0.63 of PuTTY, the best telnet / SSH client in the world.

OpenVPN Community Downloads - Free VPN client for Windows. Linux and Macs generally come with clients built-in into their network managers.

7-Zip - 7-Zip is open source software. Most of the source code is under the GNU LGPL license. The unRAR code is under a mixed license: GNU LGPL + unRAR restrictions.

Mac


FileZilla - Free FTP client that runs on  all major platforms, including Mac. Supports transferring multiple files at the same time

Fugu - Free SFTP, OpenSSH & FTP client for Mac

Fetch - paid FTP and SFTP client for Mac. Has a 15 day Free trial period.

Transmit - paid FTP and SFTP client for Mac.

CuteFTP Pro - paid FTP and SFTP client for Mac OSX.

Linux


p7zip - Free archiving client for all major platforms. It can handle .zip and .rar files, as well as its own format - .7z
Also available - 7-zip Portable.

LFTP - LFTP is a sophisticated ftp/http client, and a file transfer program supporting a number of network protocols. Like BASH, it has job control and uses the readline library for input. It has bookmarks, a built-in mirror command, and can transfer several files in parallel. It was designed with reliability in mind.

IRC Chat Programs


Chatzilla - Free IRC client for Windows, which runs as an add-on in Firefox.
Also available - Chatzilla Standalone - will run without a Firefox session running.

HexChat - formerly xchat - A Free cross platform irc client. Nice interface and easily theme-able.

MiRC - Shareware IRC client for Windows.

I-N-V-I-S-I-O-N (MiRC) - advanced feature rich all around script for mIRC

Android


Transdroid - Free - Transdroid is an Android  remote client for your BitTorrent application running on a server or home computer. Currently Transmission, ÂµTorrent, BitTorrent 6+ and Deluge 1.2+ are supported. It can show the active torrents, pause, resume or remove them and new torrents can be added via URL.

ConnectBot - Free - ConnectBot is a powerful open-source Secure Shell (SSH) client. It can manage simultaneous SSH sessions, create secure tunnels, and copy/paste between other applications.

JuicceSSH - The all in one terminal client for Android including SSH, Local Shell, Mosh and Telnet support.

OpenVPN Connect - OpenVPN Connect is the official full-featured Android VPN client for the OpenVPN Access Server, Private Tunnel and OpenVPN Community, developed by OpenVPN Technologies, Inc.

AndChat - AndChat is a multi-server internet relay chat (irc) app for Android, designed to work on both phones and tablets, and includes a tablet optimized layout.
